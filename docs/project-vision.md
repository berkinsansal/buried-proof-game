# Buried Proof

## Project Vision

Buried Proof is a narrative-driven whodunit detective game inspired by classic mystery storytelling.

Players analyze evidence, investigate suspect profiles, and reconstruct events in order to determine who committed the crime.

The experience focuses on deduction, hidden motives, conflicting perspectives, and layered clues.

## Technical Goal

The project serves as a playground for experimenting with modern AI-assisted development workflows, including structured prompting, agent-style collaboration, and AI-supported code generation.

## Architecture Philosophy

The first version will prioritize:
- simplicity
- maintainability
- small, deliverable scope

The project will be designed so it can later evolve into a larger investigation game with additional cases and features.
