# MVP Scope

## Included in Version 1

- Single playable investigation case
- Evidence exploration
- Suspect profiles
- Event timeline
- Final accusation flow
- Local game state persistence

## Excluded from Version 1

- Multiplayer
- Authentication
- Backend services
- Online progression sync
- Procedural case generation

## Content Scope

- 1 case
- 3–5 suspects
- 10–15 pieces of evidence
- 1 correct solution
